# Nexus Commands

Quick reference / description of Nexus commands. All of these command are for use in the in-game chat, or (if configured) via discord.

## Admin Commands

!nexus switch [Byte serverID]

    Switch Servers (admin only)

## User Commands

!nexus sectors [Boolean AddGps]

    Lists all sectors in the current world.

!nexus getsectors

    obsolete command: use !nexus sectors

!nexus lobby 

    Sends you back to the lobby server

!nexus nearestsector 

    Gets the Nearest Sector

!nexus onlineservers 

    Shows a list of online servers

!nexus refreshcharacter 

    Refreshes Character

!nexus info 

    Gets Server/Sector Info

!d link [Int32 key]

    Links your ingame steamID to the backend database. (Run discord command first)

!d factionlink 

    Links your faction's chat to discord
